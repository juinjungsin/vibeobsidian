# 10_Inbox — 자료 입수

## 역할
사용자가 직접 파일을 넣는 **유일한** 폴더.
AI는 이 폴더를 스캔해 INGEST 절차를 수행하고, 처리 후 원본을 삭제한다.

## 규칙
- 사용자만 파일을 넣는다
- AI는 이 폴더에 파일을 생성하지 않는다
- 처리 완료 후 원본은 자동 삭제된다 (20_Raw에 MD 원본 보존)

## 지원 형식
PDF, PPTX, DOCX, Excel, HTML, 이미지, MD 등 모든 형식.
MD가 아닌 파일은 INGEST 시 자동으로 MD로 변환된다.

## 사용법
파일을 이 폴더에 넣은 후 `/ingest` 를 입력하면 처리가 시작된다.
