# 40_Query — 질의응답 기록

## 역할
`/query` 명령으로 생성된 질문·답변의 **임시 축적 공간**.
Wiki에 승격되기 전까지 여기에 보관된다.

## 규칙
- 탐색 근거는 `30_Wiki`(필요시 `20_Raw`)로만 한정한다
- 외부 지식·인터넷으로 답하지 않는다
- 근거가 없으면 "위키에 근거가 없습니다"라고 답한다

## 프론트매터
`origin: query-answer`, `confirmed: false`

## 승격 절차
사용자가 "승격해줘"라고 지시하면 **AI가** `30_Wiki`에 직접 생성·갱신한다.
사용자는 승격 지시만 하며, Wiki를 직접 편집하지 않는다.

## 파일 네이밍
`Q_[주제]_[YYYY-MM-DD].md`
