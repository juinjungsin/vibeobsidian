# 52_Empathy_Map — 공감 지도

## 역할
사용자(고객)의 감정·행동·생각·고충을 시각적으로 정리하는 작업 공간.
Wiki의 Entity(사용자·페르소나) 노트를 근거로 작성한다.

## 파일 네이밍
`EmpathyMap_[페르소나명]_[YYYY-MM-DD].md`

## 표준 구조
```
## Think & Feel (생각과 감정)
## See (보는 것)
## Hear (듣는 것)
## Say & Do (말과 행동)
## Pain (고충)
## Gain (기대 이익)
```

## 규칙
- Wiki Entity 노트와 위키링크로 연결한다
- 완성 후 `60_Output`으로 복사 (원본 유지)
