# 50_Project — 프로젝트 작업 공간

## 역할
Wiki(30_Wiki)의 정제된 지식을 바탕으로 실제 산출물을 만드는 **응용 계층**.
사용자가 직접 검토·수정·보완할 수 있는 폴더다.

## 처리 흐름 내 위치
`Inbox → Raw → Wiki → **Project** → Output`

## 하위 폴더

| 폴더 | 역할 |
|---|---|
| `51_PCD/` | 개념설계 (Product Concept Design) |
| `52_Empathy_Map/` | 공감 지도 (사용자 감정·행동 분석) |
| `53_Customer_Journey_Map/` | 고객 여정 지도 |

새 방법론이 필요하면 `5X_` 번호 체계로 하위 폴더를 추가한다.

## 규칙
- Wiki 노트를 근거로 삼아 위키링크로 출처를 명시한다
- 사용자가 직접 내용을 수정·보완할 수 있다
- AI는 사용자의 명시적 지시가 있을 때 노트를 생성한다
- 완성된 산출물은 `60_Output`으로 복사한다 (Project 원본 유지)
