# 51_PCD — 개념설계 (Product Concept Design)

## 역할
제품·서비스의 초기 개념을 정의하는 작업 공간.
Wiki의 Concept·Entity 노트를 근거로 개념설계 문서를 생성한다.

## 파일 네이밍
`PCD_[프로젝트명]_[YYYY-MM-DD].md`

## 프론트매터
```yaml
---
title:
type: project
project:
created: YYYY-MM-DD
status: 초안 | 검토중 | 확정
related: []
---
```

## 규칙
- Wiki 근거 없이 작성하지 않는다
- 완성 후 `60_Output`으로 복사 (원본은 여기 유지)
