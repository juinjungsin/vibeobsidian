description: 30_Wiki 안에서만 근거를 찾아 답변하고 40_Query에 기록
---

다음 질문에 답하라: $ARGUMENTS

절차:
1. 근거 탐색 범위는 30_Wiki(필요시 20_Raw)로 한정한다. 외부 지식·인터넷·일반 상식으로 답하지 않는다. 근거가 없으면 "위키에 근거가 없습니다"라고 답하고 추측하지 않는다.
2. 답변과 근거 노트 목록을 함께 제시한다.
3. 질문과 답변을 40_Query에 노트로 기록한다 (origin: query-answer, confirmed: false).
4. 사용자가 승격을 지시하면 AI가 30_Wiki에 신규 노트로 생성하거나 기존 노트를 업데이트하고, origin과 confirmed(true)를 기록한다. 사용자는 Wiki를 직접 편집하지 않는다.
