# 30_Wiki — 정제된 지식 베이스

## 역할
Raw 원본을 의미 단위로 분해한 **AI 자동 생성 지식 노트**.
/query 의 탐색 대상이며, Vault의 핵심 지식 저장소다.

## 하위 분류

| 폴더 | 기준 | 예시 |
|---|---|---|
| `Concept/` | 만질 수 없는 추상 개념 | 리더십, 블루오션, 디자인씽킹 |
| `Entity/` | 사람·회사·제품 실체 | 스티브 잡스, 애플, 아이폰 |
| `Guide/` | 방법·절차 | 인터뷰 방법, OKR 설정 절차 |
| `MOC/` | 목차·지도 노트 | 디자인씽킹 전체 MOC |

## 쓰기 권한

| 주체 | 권한 |
|---|---|
| AI (/ingest) | 노트 생성·갱신 가능 |
| AI (승격) | Query→Wiki 승격 시 생성·갱신 가능 |
| 사용자 | **직접 편집 금지** |

## 규칙
- 프론트매터 없는 노트는 만들지 않는다
- 같은 개념의 노트가 있으면 새로 만들지 않고 기존 노트에 누적한다
- 관련 노트끼리 위키링크(`[[ ]]`)로 연결한다
- 관련 노트가 3개 이상이면 MOC 노트를 생성·갱신한다
